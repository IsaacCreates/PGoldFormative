Functionality

One key element of most fitnes apps is that they are easy to use and easily implementable in ones daily routine. This results a minimal starting hurdle to provide an easy start when first using the app. The apps collect the users data by requesting inputs, the smartphones sensors and having a wide compability with health gadgets like smartwatches. One Key aspect is the setting of goals. Wether it is loosing weight, gaining muscles or wanting to be able to run longer. Most apps allow the users to set goals which is a central element in keeping motivated troughout the trainings. Using this data the app provides the user fitting training plans to ease one into their first trainings or further enhance their fitness.
